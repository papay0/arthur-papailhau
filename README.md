# ArthurGPT

My personal portfolio designed to look and feel exactly like ChatGPT. Visitors "chat" with ArthurGPT to learn about my experience, projects, and skills.

**Live:** [arthur-papailhau.com](https://arthur-papailhau.com)

## Tech Stack

Next.js 15 · Tailwind CSS · shadcn/ui · TypeScript · Vercel

## Run Locally

```bash
npm install && npm run dev
```
